# Notes on David Lewis (1969) - Chapter IV: Convention and Communication

### 1. SIGNALING PROBLEMS
*   **The Model:** A communicator and an audience.
*   **Structure:**
    1.  Exactly one of several states of affairs ($s_1, s_2...$) holds.
    2.  The communicator sees which one holds.
    3.  The communicator does a "signal" ($o_1, o_2...$).
    4.  The audience sees the signal and does a "response" ($r_1, r_2...$).
*   **Coordination:** We succeed if the response matches the state of affairs (e.g., Paul Revere warns of a "sea" attack if and only if the sexton hangs two lanterns).

### 2. SIGNALING SYSTEMS
*   A **Signaling System** is a combination of two contingency plans:
    1.  *Communicator's Plan:* Maps states to signals (e.g., "If Sea, then 2 lanterns").
    2.  *Audience's Plan:* Maps signals to responses (e.g., "If 2 lanterns, then warn of Sea").
*   **Equilibria:** There are many possible systems ($n! / (n-m)!$). Any one we pick is an equilibrium because if we've settled on one, neither has an incentive to change alone.

### 3. MEANING OF SIGNALS
*   **Indicative Signals:** "The redcoats are coming by sea." These inform the audience of a state of affairs.
*   **Imperative Signals:** "Fire!" or "Stop!" These tell the audience to perform a response.
*   **Truth Conditions:** An indicative signal is "true" if the state of affairs it is mapped to actually holds. An imperative signal is "true" (or "obeyed") if the response it commands is actually performed.

### 4. MEANING-NN (GRICE)
*   Lewis links his theory to H.P. Grice’s "Non-Natural Meaning."
*   **Definition:** I mean something by a signal if I intend the audience to recognize my intention and if I intend that recognition to be their reason for acting.
*   **Conclusion:** This complex intentional state is a *consequence* of conventional signaling. If we have a convention, we don't need to reinvent these intentions every time; they are built into the structure of the coordination.

### 5. VERBAL SIGNALING
*   Verbal expressions are ideal for signals because they are easy to produce and observe, and they are intrinsically unimportant (arbitrary).
*   A rudimentary "language" is just a set of these signaling conventions.
