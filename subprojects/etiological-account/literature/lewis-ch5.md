# Notes on David Lewis (1969) - Chapter V: Conventions of Language

### 1. BEYOND SIGNALING
*   Rudimentary signaling (Ch IV) is limited: finite sentences, no "idle conversation," no general facts.
*   **Possible Language:** Lewis defines a language as a function that assigns sets of "interpretations" (mood + truth condition) to verbal expressions.
*   **Analyticity:** A sentence is "analytic" in language $L$ if its truth condition is the set of all possible worlds (it is true no matter what).

### 2. GRAMMARS
*   Languages usable by humans must be finitely specifiable.
*   **Structure:** Lexicon (words) + Generative component (rules for combining) + Representational component (producing sounds/marks).
*   **Meaning of Parts:** The meaning of a word is its "extension" in every possible world. This determines how it contributes to the truth conditions of the sentences it belongs to.

### 3. CONVENTION OF TRUTHFULNESS
*   What makes a possible language $L$ the *actual* language of a population $P$?
*   **Lewis's Contentious Claim:** It is a **Convention of Truthfulness and Trust** in $L$.
    *   *Communicators* try to utter only sentences that are true in $L$.
    *   *Audiences* respond on the assumption that communicators are trying to be truthful.
*   **Arbitrariness:** This is a convention because we could have settle on a *different* language $L'$ and been truthful/trusting in *that* one instead.

### 4. MOODS
*   **Indicative:** The communicator tries to be truthful.
*   **Imperative:** The audience tries to make the sentence true (obedience).
*   **Interrogative:** Questions are just imperatives to provide a truthful answer (e.g., "Is it raining?" = "Tell me whether it is raining").
*   **Commissive (Promises):** The communicator tries to make the sentence true *later* (fulfillment).

### 5. UNSHARP ANALYTICITY
*   Analyticity in natural language is often "fuzzy."
*   **Reason:** We don't have a convention to use *one* exact possible language. We have a convention to use *any* language from a "tight cluster" of very similar languages.
*   **Result:** A sentence is analytic if it's true in all possible worlds in *all* the languages in our cluster. If it's true in some and false in others, it's "partially analytic." This flexibility makes language easier to learn and use.
