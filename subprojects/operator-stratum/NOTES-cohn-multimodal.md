## Relevance to "The Operator Stratum"

### 1. Visual Operators: Extending the Framework

The operator stratum paper argues that grammaticality judgments target **operator value**—form–value relations that function as public coordination infrastructure. Cohn's multimodal framework implies:

- **Visual languages have operators too:** In comics, constructions like speech balloons, panel borders, and motion lines conventionally contribute to "public update" (who is speaking, temporal sequencing, character movement). These are closed paradigms, analogous to clause-structure operators in speech.
- **Diagnostics should transfer:** Paradigmatic closure, broad scope, high opportunity mass, repair sensitivity—all should apply to visual operators.

**Example:** In comics, the distinction between a rectangular panel border (default narrative time) and a wavy/scalloped border (dream/memory) is a clause-typing operator. Violating the convention (e.g., using dream borders for real-time action without context) should trigger sharp rejection—testable via visual acceptability ratings.

### 2. Phonological Operators: Cohn's Parallel

The paper already notes that phonology and lexicon CAN enter the operator stratum when grammaticalized (tone, question intonation, sign-language nonmanuals). Cohn's framework adds:

- **Graphics can be operators:** Reduplication in visuals (showing the same character twice to indicate plurality or repeated action) is iconic but conventionalized. If it functions with paradigmatic closure and high opportunity mass, it's an operator.
- **Cross-modal parity:** The principle of equivalence means that the operator–indexical distinction should apply uniformly across modalities.

### 3. Predictions for Visual Research

If the operator-stratum framework is correct, visual languages should show:

1. **Repair asymmetries:** Corrections of visual operator mismatches (wrong balloon type) should be more obligatory than corrections of stylistic choices (line thickness).
2. **Satiation resistance:** High-opportunity-mass visual gaps (e.g., impossible panel transitions) should resist satiation, just like LBE in English.
3. **ERP dissociation (speculative):** If visual comprehension can be probed with ERP, operator violations might produce different signatures than semantic/stylistic anomalies.

### 4. Information-Theoretic Angle

The paper frames operators as "protocol headers" reducing entropy. Cohn's claim that different modalities have different semiotic strengths (speech: symbolicity; graphics: iconicity; body: indexicality) suggests:

- **Visual operators are efficient** precisely because graphics are good at iconicity: the signaling channel is low-cost.
- **The entropy-reduction function is modality-neutral**, but the cost structure varies by channel—relevant to the information-theoretic framing in §5.
