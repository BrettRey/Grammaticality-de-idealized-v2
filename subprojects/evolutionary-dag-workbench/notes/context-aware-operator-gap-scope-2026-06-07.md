# Context-Aware Operator-Gap Scope Note

**Date:** 2026-06-07
**Target:** `graphs/archive/context-aware-operator-gap-candidate.json`

## Purpose

This candidate is a scoped module for cases where operator value, opportunity mass, recoverability,
constructional analogy, repair pressure, and context-sensitive judgment all matter.

It is not a replacement for the dynamic/context module. It does not model diachronic stabilization
or usage/norm feedback across time.

## Intended Stress Cards

- `left-branch-extraction`
- `transparent-free-relatives`
- `rare-uncontroversial-form`
- `grammatical-but-hard`

## Explicit Non-Goals

- Do not use this graph as a general grammaticality model.
- Do not let `operator_value` define all licensing.
- Do not treat recoverability as licensing.
- Do not treat absence as evidence without opportunity mass.
- Do not treat task-framed reported acceptability as direct grammaticality.

## Required Next Test

Create a protocol-bound evaluation for this candidate before any score changes. The evaluation
should include at least:

- a left-branch extraction case where meaning is recoverable but repair/paraphrase pressure is high;
- a transparent free-relative case where rarity is not automatically ungrammaticality;
- a rare uncontroversial form where opportunity mass is low;
- a processing-heavy case where low acceptability is not licensing failure.
