# Phenomenon Card: clause type and force

**Stress test:** Clause type, uptake role, and minor-clause boundaries.

## Description

Clause type supplies a contrast space where form can help determine what kind of move an utterance
counts as. Exclamatives, imperatives, interrogatives, declaratives, and minor clauses also show that
the boundary is not simply "finite full clause vs fragment."

## Source Pointers

- Source IDs: `cgel-clause-type`, `cgel-exclamatives-minor-types`

## Why It Matters

The case tests whether a graph can represent update-role contrasts without forcing all clause-like
or fragmentary forms into one global sentence-grammar bin.

## Minimum Contrast Cells

- major clause type with clear response expectation;
- exclamative or exclamation with overlapping affective value;
- minor clause licensed by interactional or genre context;
- form understood as content but misclassified as the wrong move type.

## Candidate Nodes

- operator_value
- register_genre_appropriateness
- genre
- medium
- semantic_pragmatic_recoverability
- repair_reformulation_pressure
- grammaticality_attribution

## Graph Failure Modes

- Treats all clause type contrasts as autonomous syntax.
- Treats minor clauses as mere performance fragments.
- Collapses uptake-role mismatch into semantic recoverability.
- Cannot represent genre-conditioned clause licensing.

## Predicted Discriminators

- An operator-oriented graph should predict repair or response-space disruption when the wrong
  move type is signalled.
- A context graph should predict why a minor clause can be licensed in one activity type and
  rejected in a formal sentence task.
