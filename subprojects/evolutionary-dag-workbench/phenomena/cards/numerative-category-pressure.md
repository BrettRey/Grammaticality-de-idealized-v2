# Phenomenon Card: numerative category pressure

**Stress test:** Numeral/numerative category, function, and licensing.

## Description

Numerals, number words, determinatives, fused heads, and countability patterns create cases where
category analysis and constructional licensing can be pulled apart. A graph should not assume that
the correct descriptive category is itself a speaker-facing licensing construct.

## Source Pointers

- Source IDs: `cgel-number-quantification`, `reynolds-numeratives`

## Why It Matters

This card supplies a category-discipline test parallel to the `more`/`less` card, but in a domain
where number, quantification, and nominal syntax interact.

## Minimum Contrast Cells

- numeral in a determiner-like nominal function;
- numeral or quantifier in fused-head use;
- countability-sensitive nominal environment;
- analyst category judgment separated from ordinary acceptability judgment.

## Candidate Nodes

- constructional_analogy
- measurement_task
- task_framing
- semantic_pragmatic_recoverability
- community_licensing
- grammaticality_attribution
- reported_acceptability

## Graph Failure Modes

- Treats category labels as causal variables.
- Collapses countability, quantity meaning, and licensing.
- Ignores the difference between analyst classification tasks and speaker acceptability tasks.
- Treats constructional analogy as raw frequency.

## Predicted Discriminators

- A measurement-aware graph should predict category-task/acceptability-task divergence.
- A constructional graph should predict which numerative patterns extend by analogy and which
  remain tied to specific nominal environments.
