# Phenomenon Card: interjection boundary

**Stress test:** Uptake configuration without automatic operator-stratum membership.

## Description

Interjections and exclamations can configure uptake, repair, stance, attention, floor management,
or affect. Many tokens are also stretchable, repeatable, blendable, and open to innovation, which
makes them a boundary case for any graph that links public consequence to grammaticality.

## Source Pointers

- Source IDs: `cgel-exclamatives-minor-types`, `reynolds-interjections`

## Why It Matters

This card tests whether graphs can represent public uptake effects without classifying every
uptake-configuring form as a closed operator contrast.

## Minimum Contrast Cells

- conventional repair-initiating interjection;
- stretched or repeated affective interjection;
- interjection-like form in a genre with stable conventions;
- invented token whose stance is understood but whose form is not conventional.

## Candidate Nodes

- operator_value
- social_indexical_value
- register_genre_appropriateness
- production_probability
- community_licensing
- semantic_pragmatic_recoverability
- repair_reformulation_pressure

## Graph Failure Modes

- Treats all uptake configuration as operator value.
- Treats all interjections as payload or noise.
- Ignores openness and token-level innovation.
- Collapses stance reaction with grammaticality attribution.

## Predicted Discriminators

- An operator graph should identify only the interjection-like cases that enter a restricted,
  accountable repertoire.
- A social-indexical graph should predict stance reactions without turning them into structural
  licensing failures.
